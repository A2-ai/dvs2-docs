**dvs** is built and maintained by [**A2-AI**](https://a2-ai.com).

A2-AI is a scientific consulting firm working in pharmacometrics, clinical
pharmacology, and regulatory strategy for drug development. It builds custom
technology and GxP-compliant cloud computing for pharmaceutical and biotech
clients.

dvs grew out of that work. Drug-development programs generate large, sensitive
datasets that have to be versioned, shared, and reproduced exactly, but do not
belong in a Git history. dvs is the independent data version control system
A2-AI built to handle them, usable with Git or on its own.

- Website: [a2-ai.com](https://a2-ai.com)
- Source: [github.com/A2-ai/dvs2](https://github.com/A2-ai/dvs2)
- Based in Ann Arbor, Michigan
